# Portfolio - Danilo Cáceres

Portfolio personal construido con [Astro](https://astro.build/), [Tailwind CSS](https://tailwindcss.com/) y [React](https://react.dev/). Soporte multiidioma (Español/Inglés), modo oscuro/claro y diseño responsivo.

## 🚀 Stack

- **Astro 4** - Framework de construcción de sitios web
- **Tailwind CSS** - Estilos utilitarios
- **React** - Componentes interactivos
- **TypeScript** - Tipado estático
- **i18n** - Sistema de internacionalización propio

## 🛠️ Comandos

| Comando | Acción |
|---------|--------|
| `npm run dev` | Inicia servidor de desarrollo en `localhost:4321` |
| `npm run build` | Build de producción (`./dist/`) |
| `npm run preview` | Previsualiza el build localmente |
| `npm run astro` | CLI de Astro |

## 🌐 Estructura

```
src/
├── components/   # Componentes reutilizables
├── i18n/         # Traducciones ES/EN
├── layouts/      # Layout principal
├── pages/        # Rutas del sitio
└── env.d.ts      # Tipados globales
```
